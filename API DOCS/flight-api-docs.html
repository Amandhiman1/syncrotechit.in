Flight API Documentation

POST /api/reports/flights-by-destination-type

Description: Fetch flights by destination type (city, country, or continent).

Request Body:

 {
     "query": {
       "%city2%": "Mumbai",
       "%country2%": "India",
       "%continent2": "Asia"
     },
     "flightsNumber": 5,
     "lang": "en"
   }

Response:

{
    "flights": [...],
    "queryString": {...}
}

POST /api/reports/flights-to-city

Description: Fetch one-way or return flights to a specified city.

Request Body:

{
    "query": {
        "%city%": "Delhi",
        "%city2%": "Mumbai"
    },
    "flightsNumber": 5,
    "lang": "en",
    "type": "one-way"
}

Response (One-Way):

{
    "flights": [...],
    "queryString": {...}
}

Response (Return):

{
    "flights": {
        "outboundFlights": [...],
        "returnFlights": [...]
    },
    "queryString": {...}
}
